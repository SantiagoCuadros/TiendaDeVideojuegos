# Tienda de Videojuegos

Sistema de consola en Python para administrar el inventario y las ventas de una tienda de videojuegos. Permite agregar, buscar, actualizar, vender y eliminar productos,

además de mostrar estadísticas e historial de ventas, aplicando variables, condicionales, ciclos, funciones y colecciones (listas y diccionarios).
